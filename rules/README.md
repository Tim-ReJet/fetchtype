# Rules

Current MVP validation rules live in `@fetchtype/core` and focus on practical adoption:

- Minimum readable contrast for primary text against primary background
- Minimum body line-height
- Minimum button font size
- Maximum prose width guidance

Future policy packs can extend this directory with machine-readable rule definitions and documentation for:

- Multi-brand theme policies
- Locale and script-specific typography
- Variable font axis constraints
- Team-specific approval and exception workflows
